# デフォルト動作ルール

## スキル起動時の確認

ユーザーがスキルを起動したとき（例: 「瞬間英作文」「ディクテーション」など）、**必ず最初に対応する `SKILL.md` を読んでから**練習を開始すること。

利用可能なスキル一覧:

| スキル名 | ディレクトリ | 概要 |
|---|---|---|
| free-conversation | `.kiro/skills/free-conversation/SKILL.md` | 英語メンターとの自由英会話・添削 |
| instant-translation | `.kiro/skills/instant-translation/SKILL.md` | 日本語→英語の瞬間英作文 |
| dictation-practice | `.kiro/skills/dictation-practice/SKILL.md` | 英文の聞き取りディクテーション |
| vocabulary-builder | `.kiro/skills/vocabulary-builder/SKILL.md` | 文脈付き語彙トレーニング |
| grammar-fix | `.kiro/skills/grammar-fix/SKILL.md` | 文法エラー発見・修正トレーニング |
| situation-roleplay | `.kiro/skills/situation-roleplay/SKILL.md` | シチュエーション別ロールプレイ |
| email-writing | `.kiro/skills/email-writing/SKILL.md` | ビジネスメール・Slackライティング |
| shadowing | `.kiro/skills/shadowing/SKILL.md` | シャドーイング練習 |

SKILL.md を読まずに練習を始めてはいけない。
