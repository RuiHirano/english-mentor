# english-mentor

Kiro CLI のスキルを使った英語学習ツール。音声読み上げ付きで英会話・英作文の練習ができます。

## スキル一覧

### English Mentor（英会話練習）

英語メンターと自由に英会話するスキル。

- 英語で話しかければ英語で返答、日本語なら日本語で説明＋英語例文
- 文法ミスや不自然な表現をやさしく添削
- メンターの英語発話を `say` コマンドで音声読み上げ（Samantha, 160wpm）

### Instant Translation（瞬間英作文）

日本語の文を英訳する瞬間英作文トレーニング。

- 難易度3段階（初級・中級・上級）
- 日常会話・ビジネス・旅行など多様なシチュエーションから出題
- ⭕/🔺/❌ の判定＋模範解答＋解説
- 模範解答を `say` コマンドで音声読み上げ
- スコア管理（正答率の追跡）

特別コマンド:
- `レベル変更: 初級/中級/上級` — 難易度変更
- `テーマ: テーマ名` — 特定テーマに絞って出題
- `もう一回` — 同じ問題を再出題
- `解説詳しく` — 直前の問題を詳しく解説
- `スコア` — 正答率を表示

### Dictation Practice（ディクテーション）

英文を音声で読み上げ、聞き取った内容をタイプして答えるリスニング練習。

- 難易度3段階（初級・中級・上級）
- 音声のみ再生→ユーザーが書き取り→正解と比較
- リンキング・弱形など聞き取りポイントの解説付き
- `ゆっくり` コマンドで速度を落として再生可能

### Vocabulary Builder（語彙力トレーニング）

文脈の中で英単語を学ぶ語彙力トレーニング。

- 穴埋めモード（英文の空欄を埋める）と日→英モード（日本語→英語）を交互に出題
- 単語だけでなく句動詞・イディオムも出題
- 類義語・反義語・使い分けのポイントを補足

### Grammar Fix（文法エラー修正）

文法エラーを含む英文を見つけて修正するトレーニング。

- 時制、主語動詞の一致、冠詞、前置詞、語順など幅広い文法項目
- 1文につき1〜2箇所のエラー
- 修正理由と関連する文法ルールをワンポイント解説

### Situation Roleplay（シチュエーション別ロールプレイ）

レストラン、ホテル、面接など特定場面での英会話ロールプレイ。

- メンターが相手役（店員、面接官など）を演じる
- 会話の流れを止めずにさりげなく添削
- `ヒント` コマンドで次に言うべきフレーズを提示
- `まとめ` で使った重要フレーズを一覧表示

### Email/Slack Writing（ビジネスライティング）

ビジネスメールやSlackメッセージを書く練習。

- 会議調整、進捗報告、依頼、謝罪など実践的なお題
- フォーマル/カジュアルのトーン適切さもチェック
- 添削＋そのシチュエーションで使える定番フレーズを紹介

### Shadowing（シャドーイング）

英文を聞いて復唱する発音・リスニング練習。

- 英文テキスト＋音声を提示→ユーザーが音声入力で復唱
- 認識結果と原文を比較して発音をチェック
- リンキング・アクセントなど発音のコツを解説
- 音声認識の精度に依存するため判定は参考程度

## 学習プランナー（エージェント）

目標設定から練習プラン作成、進捗管理までを一括で行うエージェント。

```bash
./study
```

で起動すると:
1. Streamlit ダッシュボード（http://localhost:8501）がバックグラウンドで起動
2. `kiro chat --agent study-planner` で学習プランナーが起動

### 機能

- **初回セットアップ**: 目標・レベル・苦手分野をヒアリングし、学習プランを自動生成
- **セッション管理**: `workspace/YYYYMMDDHHMM.md` に練習ログを記録
- **進捗トラッキング**: `data/progress.yaml` にスコアを蓄積
- **週次レビュー**: 正答率の推移を分析し、プランを自動調整
- **リクエスト管理**: 重点項目やスキップしたいスキルを記録

### データファイル

| ファイル | 内容 |
|---|---|
| `data/profile.yaml` | ユーザーの目標・レベル |
| `data/preferences.yaml` | 好み・リクエスト |
| `data/plan.yaml` | 週次学習プラン |
| `data/progress.yaml` | 累積の進捗データ |

### ダッシュボード

Streamlit で進捗を可視化:
- スコア推移グラフ
- スキル別正答率
- 週次レビューサマリー
- 現在の学習プラン

## 音声読み上げについて

macOS の `say` コマンドで英語部分を読み上げます。

- 音声: Samantha（米国英語）
- 速度: 160wpm（少しゆっくり）

### 音声入力との併用

macOS の音声入力（Dictation）を有効にしていると、`say` の音声出力がミュートされます。再生時は音声入力を一時的にオフにしてください。

## セットアップ

[Kiro CLI](https://kiro.dev) をインストールし、このリポジトリで `kiro chat` を実行してください。
